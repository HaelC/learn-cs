---
tag: C
type: glossary
overview: Variable, function, argument, escape sequence ...
---

- variable  
  Variables store values used during the computation.
- function  
  A function contains statements that specify the computing operations to be done.
  - `main`  
    `main` function is where your program begins executing.
  - `printf`  
    `printf` is a library function that prints output.
- argument  
  Arguments are a list of values provided to communicate data between functions. Argument list can be empty.
- escape sequence  
  An escape sequence provides a general and extensible mechanism for representing hard-to-type or invisible characters.
  - `\n` for newline

These glossaries mainly come from <cite>The C Programming Language, Second Edition</cite>.
